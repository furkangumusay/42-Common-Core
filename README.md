Bu depo, 42 Türkiye temel eğitimi (Common Core) boyunca geliştirdiğim projeleri içerir. Tüm projeler sıfırdan Unix standartlarında tasarlanmış ve çeşitli algoritmik/mimari problemlerin çözümlerini içermektedir. Detaylar için projelere tıklayarak o projeye ait özel mimari dökümantasyonlarını inceleyebilirsiniz.
