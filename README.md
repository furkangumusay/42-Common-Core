# 42 Kocaeli - Common Core Projeleri

Bu depo, 42 Kocaeli temel eğitimi (Common Core) boyunca geliştirdiğim projeleri içerir. Tüm projeler C veya C++ kullanılarak sıfırdan Unix standartlarında tasarlanmış ve çeşitli algoritmik/mimari problemlerin çözümlerini içermektedir. Detaylar için projelerin kendi klasörlerine tıklayarak o projeye ait özel dökümantasyon ve mimari kararları inceleyebilirsiniz.
